# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest| :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities to the GitHub Security Advisory tab
or email maintainers directly. Do not open public issues for security bugs.

## Responsible Disclosure

We request a 90-day disclosure window before public reporting.
